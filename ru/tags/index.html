Поиск

Используется нечёткий поиск (приблизительное сопоставление строк). Это метод поиска строк, которые приблизительно соответствуют поисковому шаблону. Результаты поиска сортируются по точности совпадения с поисковым шаблоном. Результаты поиска, наиболее точно соответствующие запросу, находятся сверху, приблизительные результаты - снизу.
