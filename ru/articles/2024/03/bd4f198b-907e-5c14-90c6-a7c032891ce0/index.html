Когда требуется узнать информацию о домене, можно воспользоваться локальными утилитами для диагностики его DNS-записей.

Общая информация

Рассмотрим запросы общей информации по доменам.

Dig

Получить общую информацию по домену в Dig:

dig example.com

Информация по DNS-записям

В отдельных случаях требуется получить информацию по конкретным DNS-записям домена.

Dig

Получить информацию по MX-записи домена в Dig:

dig mx example.com

Поиск

Используется нечёткий поиск (приблизительное сопоставление строк). Это метод поиска строк, которые приблизительно соответствуют поисковому шаблону.

Результаты поиска сортируются по точности совпадения с поисковым шаблоном. Результаты поиска, наиболее точно соответствующие запросу, находятся сверху, приблизительные результаты - снизу.
