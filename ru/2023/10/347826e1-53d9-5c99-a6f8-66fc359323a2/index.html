Динамическое "(Powered by Invision Power Board)" на IP.Board 2.3

  • 2023/10/17 03:10:06 UTC
  • •

Данный материал является артефактом прошлого. Он был найден в старых работах и телепортирован сюда для сохранения в истории.

Содержание

  • Внедрение модификации

Нашёл ещё одну модификацию а-ля “прикол на зависть другим” для #IPB. Модификация позволяет сделать из копирайта форума динамически изменяемую “свистульку”.

Внедрение модификации

Открыть файл ./sources/action_public/boards.php, найти:

1

$cp = " (Powered by Invision Power Board)";

Заменить на:

1
2
3
4
5
6
7
8

$cp_phrases = array(
  '0' => ' - CYBER-CITY: Поддержка',
  '1' => ' - CYBER-CITY: Разработка',
  '2' => ' - CYBER-CITY: Исследование',
);

$cp_rand = rand(0, 2);
$cp = $cp_phrases[$cp_rand];

Не забудьте, если у вас больше фраз, чем 3, то измените rand(0, 2), где 0 - первая фраза, 2 - последняя.

Поиск

Используется нечёткий поиск (приблизительное сопоставление строк). Это метод поиска строк, которые приблизительно соответствуют поисковому шаблону.

Результаты поиска сортируются по точности совпадения с поисковым шаблоном. Результаты поиска, наиболее точно соответствующие запросу, находятся сверху, приблизительные результаты - снизу.
