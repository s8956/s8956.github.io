Дополнительная страница для vBulletin 4

  • 2023/10/17 18:01:38 UTC
  • •

Данный материал является артефактом прошлого. Он был найден в старых работах и телепортирован сюда для сохранения в истории.

Содержание

  • Что такое дополнительная страница (или Custom Pages)?
  • Как сделать дополнительную страницу?

Создание дополнительных страниц для уже устаревшего форумного движка #vBulletin 4.

Для #vBulletin версии 4.0 действия практически одни и те же, как в материале Дополнительная страница для vBulletin 3.

Что такое дополнительная страница (или Custom Pages)?

Дополнительная страница (или Custom Pages) - это страница, “совмещённая” со стилем форума.

Как сделать дополнительную страницу?

Необходимо создать шаблон custom_ШАБЛОН, содержимое которого представляет собой следующий код:

Поиск

Используется нечёткий поиск (приблизительное сопоставление строк). Это метод поиска строк, которые приблизительно соответствуют поисковому шаблону.

Результаты поиска сортируются по точности совпадения с поисковым шаблоном. Результаты поиска, наиболее точно соответствующие запросу, находятся сверху, приблизительные результаты - снизу.
