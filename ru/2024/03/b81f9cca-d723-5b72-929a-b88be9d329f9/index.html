Cisco: Обновление IOS

Инструкция по резервному копированию и обновлению #Cisco #IOS.

Общая настройка

Необходимо предварительно настроить сетевой интерфейс компьютера со следующими параметрами:

  • IP-адрес: 192.168.1.2.
  • Маска подсети: 255.255.255.0.
  • Шлюз: 192.168.1.1.
  • DNS: 8.8.8.8.

Теперь настроим сетевой интерфейс FE0/1 на #Cisco:

1
2
3
4
5
6
7

# conf t
(config)# int fa0/1
(config-if)# ip address 192.168.1.1 255.255.255.0
(config-if)# no shutdown
(config-if)# exit
(config)# exit
#

После предварительных настроек, подключим сетевой интерфейс компьютера к сетевому интерфейсу #Cisco FE0/1 и запустим TFTP-сервер на компьютере c IP 192.168.1.2.

Резервное копирование текущего образа IOS

Смотрим текущую версию #IOS:

1

# show version

На всякий случай, делаем резервную копию текущего образа #IOS на TFTP-сервер:

1
2

# show flash: | include .bin
# copy flash:IOS_FW.bin tftp://192.168.1.2

Где:

  • flash:IOS_FW.bin - источник, файл текущего образа #IOS на flash-карте (IOS_FW.bin).
  • tftp://192.168.1.2 - назначение, IP-адрес TFTP-сервера.

Загрузка нового образа IOS

Загружаем на флэш-карту новый образ #IOS:

1

# copy tftp://192.168.1.2/IOS_FW.bin flash:

Где:

  • tftp://192.168.1.2/IOS_FW.bin - источник, адрес TFTP-сервера и путь к файлу с образом #IOS (IOS_FW.bin).
  • flash: - назначение, flash-карта.

Верификация нового образа IOS

Проверяем контрольную сумму загруженного образа #IOS:

1
2

# show flash: | include .bin
# verify /md5 flash:IOS_FW.bin

Где:

  • flash:IOS_FW.bin - образ #IOS (IOS_FW.bin) на flash-карте.

Обновление IOS

Перед обновлением рекомендуется сделать резервную копию лицензий:

1

# license save flash:ALL_LICENSE.lic

Прописываем в конфигурации #Cisco загрузку нового образа #IOS и перезагружаем оборудование:

1
2
3
4

# conf t
(config)# boot system flash IOS_FW.bin
(config)# exit
# reload

Где:

  • flash:IOS_FW.bin - образ #IOS (IOS_FW.bin) на flash-карте.

Проверка новой версии IOS

После перезагрузки смотрим версию #IOS:

1

# show version

Используется нечёткий поиск (приблизительное сопоставление строк). Это метод поиска строк, которые приблизительно соответствуют поисковому шаблону.
