NodeJS: Установка

  • 2025/02/07 21:00:51 UTC
  • •

Содержание

  • Экспорт параметров
  • Репозиторий
  • Установка

Инструкция по установке и первичной настройке #NodeJS.

Экспорт параметров

  • Экспортировать заранее подготовленные параметры в переменные окружения:
1

export NODE_VER='22'

Репозиторий

  • Скачать и установить ключ репозитория:
1

curl -fsSL 'https://deb.nodesource.com/gpgkey/nodesource-repo.gpg.key' | gpg --dearmor -o '/etc/apt/keyrings/nodesource.gpg'
  • Создать файл репозитория /etc/apt/sources.list.d/nodesource.sources:
1

echo -e "X-Repolib-Name: Node.js\nTypes: deb\nURIs: https://deb.nodesource.com/node_${NODE_VER}.x\nSuites: nodistro\nComponents: main\nSigned-By: /etc/apt/keyrings/nodesource.gpg\n" | tee '/etc/apt/sources.list.d/nodesource.sources' > '/dev/null'
  • Скачать файлы предпочтений в /etc/apt/preferences.d/:
1

f=('nodejs' 'nsolid'); d='/etc/apt/preferences.d'; s='https://lib.onl/ru/2025/02/138218c7-d9a9-5eeb-a4cf-31ba8dc24896'; for i in "${f[@]}"; do curl -fsSLo "${d}/${i}.pref" "${s}/${i}.pref"; done

Установка

  • Установить пакеты:
1

apt update && apt install --yes nodejs

Поиск

Используется нечёткий поиск (приблизительное сопоставление строк). Это метод поиска строк, которые приблизительно соответствуют поисковому шаблону.

Результаты поиска сортируются по точности совпадения с поисковым шаблоном. Результаты поиска, наиболее точно соответствующие запросу, находятся сверху, приблизительные результаты - снизу.
