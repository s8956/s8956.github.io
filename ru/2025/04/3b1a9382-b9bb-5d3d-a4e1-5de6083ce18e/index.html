Работа с MongoDB

  • 2025/04/05 18:13:50 UTC
  • •

  • Базы данных
  • Коллекции
  • Пользователи

Шпаргалка по работе с #MongoDB.

Базы данных

  • Показать список баз данных:
1

show dbs
  • Создать (использовать) базу данных DB_NAME:
1

use DB_NAME

Коллекции

  • Показать коллекции:
1

show collections
  • Создать коллекцию COLL_NAME:
1

db.createCollection("COLL_NAME")
  • Удалить коллекцию COLL_NAME:
1

db.COLL_NAME.drop()

Пользователи

  • Создать пользователя DB_USER с паролем PASSWORD:
1

db.createUser({user: "DB_USER", pwd: "PASSWORD", roles: []})
  • Создать пользователя DB_USER с паролем PASSWORD и присвоить ему роль readWrite для базы данных DB_NAME:
1

db.createUser({user: "DB_USER", pwd: "PASSWORD", roles: [{role: "readWrite", db: "DB_NAME"}]})
  • Удалить пользователя DB_USER:
1

db.dropUser("DB_USER")

Поиск

Используется нечёткий поиск (приблизительное сопоставление строк). Это метод поиска строк, которые приблизительно соответствуют поисковому шаблону.

Результаты поиска сортируются по точности совпадения с поисковым шаблоном. Результаты поиска, наиболее точно соответствующие запросу, находятся сверху, приблизительные результаты - снизу.
